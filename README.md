# 🌍 Geotrack-live

## 📌 Project Overview
Geotrack-live is a **real-time multi-user location tracking system** that allows multiple users to share and view each other's live locations on a map.

It is built using modern web technologies like **Node.js, Socket.IO, and Leaflet.js**, enabling fast and real-time communication between users.

---

## 🚀 Features

- 📍 Real-time location tracking  
- 👥 Multi-user support  
- 🗺️ Interactive map using Leaflet  
- 🔄 Live updates using Socket.IO  
- 🟢 Online / Offline user status  
- 🔐 User session handling  
- 👨‍👩‍👧 Group or room-based tracking  
- 📡 Automatic location updates
